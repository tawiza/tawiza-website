À propos

Tawiza rend les données publiques françaises lisibles. On collecte, on croise, on raconte. Pas un SaaS, pas un produit - un bien commun numérique.

Le projet est open source, les données sont publiques, les analyses sont gratuites. Si ça vous parle, vous êtes au bon endroit.

Qui

Tawiza est porté par un contributeur unique pour l'instant. Le code est ouvert, les analyses sont sourcées, les méthodes sont documentées. L'identité du mainteneur n'est pas publique — ce qui l'est, c'est le travail. Jugez le travail, pas le CV.

Le « on » que vous lisez sur ce site, c'est un pari : celui qu'il deviendra vrai. Si le projet vous parle, il peut devenir collectif. C'est tout l'objet de la page Contribuer.

Où en est le projet

Tawiza est en phase de construction. Les analyses publiées sont réelles — chaque chiffre est sourcé, chaque source est vérifiable. L'outil technique (collecte automatisée, agent IA, dashboard) est en développement. Ce qui est publié fonctionne. Ce qui ne fonctionne pas encore n'est pas publié.

Pourquoi « Tawiza »

Tawiza (ⵜⴰⵡⵉⵣⴰ) vient de l'amazigh. C'est une forme d'entraide collective pratiquée dans les communautés amazighes. Quand quelqu'un a besoin de construire sa maison, de récolter ses champs ou de préparer un événement, la communauté se mobilise. Pas de contrat, pas de rémunération - juste la certitude que quand tu auras besoin, les autres seront là.

On a trouvé que ça résumait bien l'idée : construire ensemble quelque chose que personne ne pourrait faire seul.

Trois principes

Montrer, pas promettre

Pas de roadmap à 5 ans, pas de « bientôt ». Ce qui est publié fonctionne, ce qui ne fonctionne pas n'est pas publié.

Sourcer tout

Chaque chiffre, chaque graphique a sa source. Vérifiable en un clic. Si on ne peut pas le prouver, on ne le dit pas.

Rester ouvert

Le code est sur GitHub. Les données sont publiques. Les méthodes sont documentées. Un bien commun ne peut pas être une boîte noire.

Transparence & télémétrie

Tawiza collecte des données de télémétrie anonymes pour améliorer la plateforme. C'est opt-out — vous pouvez désactiver à tout moment.

Ce qui est collecté

Version de Tawiza, système d'exploitation, fonctionnalités utilisées, sources de données interrogées, durée des requêtes agents. Un identifiant anonyme (hash aléatoire) est généré localement, sans aucune information personnelle.

Ce qui n'est PAS collecté

Aucune adresse IP, aucun email, aucun nom, aucun contenu de chat ou de requête, aucune donnée d'entreprise ou territoriale, aucun token ou clé API.

Infrastructure

PostHog hébergé en UE (eu.i.posthog.com). Clé write-only (ne peut pas lire vos données). Rétention 90 jours. RGPD conforme. Le code de télémétrie est 100% transparent.

Pour désactiver : TELEMETRY_ENABLED=false dans votre fichier .env. C'est tout.

Le site web (tawiza.fr) utilise Plausible Analytics self-hosté — sans cookies, conforme RGPD.

Nous retrouver

Le projet vit sur GitHub. Le reste, c'est pour discuter.
